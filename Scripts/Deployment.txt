To deploy the API:
------------------------------------
1. Run the "ObsTool API - deploy & run.cmd", or

  Go to Project dir..
  dotnet publish -c Release -r win10-x64
  Go to publish dir..
  .\ObsTool.exe

Complete deployment directory:
  C:\Users\Olle\source\repos\ObsToolApi\ObsTool\bin\Release\netcoreapp2.0\


To deploy the Front-end:
------------------------------------
1. Run the "ObsTool FE - deploy.cmd" script, or

  In the FE git run:

    npm build      # To build with webpack
    npm deploy     # To copy the files to the directory used by the webserver.

2. Make sure the serving webserver is running by running "ObsTool FE - webserver.cmd", or 

  In the Web server git run: 

    cd .....\ObsToolWebserver
    npm start

Complete deployment directory:
  C:\Users\Olle\source\repos\ObsToolClientServed\build\
